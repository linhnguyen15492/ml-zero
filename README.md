# machine learning from scratch

This repository is using resources from CS229 by Andrew Ng.

This code from this repository is from Machine Learning Specialization by DeepLearning.AI and Andrew Ng on Coursera. The code is in Python and uses NumPy and Matplotlib libraries with minimal use of libraries.
The code is organized in the `src` directory, which contains the main code for the cost function and utility functions. The `README.md` file provides an overview of the repository and its contents.

Source in this repo is from scratch, so it may not be as efficient or optimized as other implementations. However, it serves as a great learning resource for understanding the fundamentals of machine learning algorithms and concepts.
